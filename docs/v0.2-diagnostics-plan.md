# v0.2 Diagnostics Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add conservative in-editor diagnostics (squiggles) for `.kicad_dru` files that flag only definitely-wrong rules, never false-positiving on a rule KiCad would accept.

**Architecture:** A pure, `vscode`-free s-expression parser (`src/parser.ts`) produces an AST + structural errors; a pure validator (`src/diagnostics.ts`) walks the AST against `data/api.json` and returns diagnostics; `src/extension.ts` (the only vscode-coupled file) wires a debounced `DiagnosticCollection`. Mirrors the existing completion/hover pure-function + adapter split.

**Tech Stack:** TypeScript, esbuild (bundle + ESM test builds), `node --test` for pure unit tests, `vscode-tmgrammar-test`/`-snap` for the (unchanged) grammar, VS Code `languages.createDiagnosticCollection`.

## Global Constraints

- Governing rule: **only flag what is definitely wrong; when unsure, emit nothing.** KiCad's *Check Rule Syntax* is the source of truth. (verbatim from spec)
- Tier A → `Error`; Tier B → `Warning`; Tier C (semantic/value-range/rule-logic) → out of scope.
- All new logic lives in pure, `vscode`-free modules under `src/`, unit-tested via `node --test`. `src/extension.ts` is the only file importing `vscode`.
- Reuse `data/api.json` as the only vocabulary source; add NO new data fields.
- `@types/vscode` stays `~1.80.0` (engine floor `^1.80.0`). esbuild externalises `vscode`.
- Every diagnostic: `source: "kicad-dru"` + a stable kebab-case `code`.
- Setting `kicad-dru.diagnostics`: `"off" | "errors" | "all"`, default `"all"`.
- `npm test` must stay green (snapshots + assertion cases + unit), and every existing valid fixture must produce ZERO diagnostics.

- [ ] **Step 3: Implement `src/parser.ts`**

Single-pass tokenizer + recursive descent. Algorithm:
- Walk the text char by char tracking `line`/`col`. At the start of each physical line, if the first non-whitespace char is `#`, skip to end of line (KiCad line-start-only comments).
- Tokens: `(` (open), `)` (close), `"`/`'`-delimited strings (single physical line; if the closing quote isn't found before newline/EOF, emit token with `terminated:false` + push `unterminated-string` error, end the string at line end), and atoms (runs of non-whitespace, non-paren, non-quote chars; `${...}` text variables are atoms).
- Parser: maintain a stack of open list nodes. `(` pushes a new list (record `open` range; the head is the first atom child's value, else `null`). `)` pops; if stack empty → `extra-close-paren` error at that range; else set the list's `close` range and `range.end`. Atoms/strings append to the current list's children (or to top-level `ast` if stack empty). At EOF, any lists still on the stack → `unclosed-paren` error (range = the open paren). `head` = first child if it's an atom, else `null`.

```ts
export interface Pos { line: number; col: number }
export interface Range { start: Pos; end: Pos }
export type Node =
  | { kind: 'list'; head: string | null; children: Node[]; range: Range; open: Range; close: Range | null }
  | { kind: 'atom'; value: string; range: Range }
  | { kind: 'string'; value: string; quote: '"' | "'"; terminated: boolean; range: Range };
export interface ParseError { code: string; message: string; range: Range }
export interface ParseResult { ast: Node[]; errors: ParseError[] }

export function parse(text: string): ParseResult {
  const errors: ParseError[] = [];
  const lines = text.split('\n');
  type Frame = { node: Extract<Node, { kind: 'list' }> };
  const stack: Frame[] = [];
  const ast: Node[] = [];
  const push = (n: Node) => (stack.length ? stack[stack.length - 1].node.children.push(n) : ast.push(n));

  for (let line = 0; line < lines.length; line++) {
    const s = lines[line];
    // line-start-only comment
    if (/^\s*#/.test(s)) continue;
    let col = 0;
    while (col < s.length) {
      const ch = s[col];
      if (ch === ' ' || ch === '\t') { col++; continue; }
      if (ch === '(') {
        const open = { start: { line, col }, end: { line, col: col + 1 } };
        const node = { kind: 'list', head: null, children: [], range: { start: { line, col }, end: open.end }, open, close: null } as Extract<Node, { kind: 'list' }>;
        push(node);
        stack.push({ node });
        col++; continue;
      }
      if (ch === ')') {
        const range = { start: { line, col }, end: { line, col: col + 1 } };
        const top = stack.pop();
        if (!top) { errors.push({ code: 'extra-close-paren', message: 'Unmatched close parenthesis.', range }); }
        else { top.node.close = range; top.node.range.end = range.end;
               const first = top.node.children[0];
               top.node.head = first && first.kind === 'atom' ? first.value : null; }
        col++; continue;
      }
      if (ch === '"' || ch === "'") {
        const startCol = col; col++;
        let value = '';
        let terminated = false;
        while (col < s.length) {
          if (s[col] === ch) { terminated = true; col++; break; }
          value += s[col]; col++;
        }
        const range = { start: { line, col: startCol }, end: { line, col } };
        if (!terminated) errors.push({ code: 'unterminated-string', message: 'Unterminated string literal.', range });
        push({ kind: 'string', value, quote: ch, terminated, range });
        continue;
      }
      // atom
      const startCol = col;
      while (col < s.length && !' \t()'.includes(s[col]) && s[col] !== '"' && s[col] !== "'") col++;
      push({ kind: 'atom', value: s.slice(startCol, col), range: { start: { line, col: startCol }, end: { line, col } } });
    }
  }
  for (const f of stack) errors.push({ code: 'unclosed-paren', message: 'Unclosed parenthesis.', range: f.node.open });
  return { ast, errors };
}
```

```ts
import { parse, type Node, type Range } from './parser';
import type { ApiData } from './completion';

export type Severity = 'error' | 'warning';
export type Level = 'off' | 'errors' | 'all';
export interface Diag { range: Range; severity: Severity; code: string; message: string }

export function computeDiagnostics(text: string, api: ApiData, level: Level = 'all'): Diag[] {
  if (level === 'off') return [];
  const { ast, errors } = parse(text);
  const out: Diag[] = [];
  // Structural parse errors are Tier A.
  for (const e of errors) out.push({ range: e.range, severity: 'error', code: e.code, message: e.message });
  // (Tasks 3-8 push more checks here, walking `ast`.)
  if (level === 'errors') return out.filter((d) => d.severity === 'error');
  return out;
}
```

```ts
const diagnostics = vscode.languages.createDiagnosticCollection('kicad-dru');
context.subscriptions.push(diagnostics);
const SEV = { error: vscode.DiagnosticSeverity.Error, warning: vscode.DiagnosticSeverity.Warning };
const level = () => vscode.workspace.getConfiguration('kicad-dru').get<'off'|'errors'|'all'>('diagnostics', 'all');
const validate = (doc: vscode.TextDocument) => {
  if (doc.languageId !== 'kicad-dru') return;
  const ds = computeDiagnostics(doc.getText(), data, level()).map((d) => new vscode.Diagnostic(
    new vscode.Range(d.range.start.line, d.range.start.col, d.range.end.line, d.range.end.col),
    d.message, SEV[d.severity]));
  for (const x of ds) x.source = 'kicad-dru';
  // attach codes
  ds.forEach((x, i) => { x.code = computeDiagnostics(doc.getText(), data, level())[i]?.code; });
  diagnostics.set(doc.uri, ds);
};
const timers = new Map<string, NodeJS.Timeout>();
const schedule = (doc: vscode.TextDocument) => {
  const key = doc.uri.toString();
  clearTimeout(timers.get(key)); timers.set(key, setTimeout(() => validate(doc), 300));
};
context.subscriptions.push(
  vscode.workspace.onDidOpenTextDocument(validate),
  vscode.workspace.onDidChangeTextDocument((e) => schedule(e.document)),
  vscode.workspace.onDidCloseTextDocument((doc) => diagnostics.delete(doc.uri)),
  vscode.workspace.onDidChangeConfiguration((e) => { if (e.affectsConfiguration('kicad-dru.diagnostics')) vscode.workspace.textDocuments.forEach(validate); }),
);
vscode.workspace.textDocuments.forEach(validate);
```
